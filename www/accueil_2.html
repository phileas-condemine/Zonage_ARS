Sources des données

Les contours géographiques sont récupérés via l'API geo.api.gouv.fr, un service fourni par Etalab, qui s'appuie sur le code officiel géographique le plus récent. La "date de dernière mise à jour des contours géographiques" affichée dans l'application sous les jauges indique la dernière date de collection des données à partir de l'API geo.api.gouv.fr.

Les données de population sont puisées dans différentes sources selon la profession de santé et la zone géographique.

  • Pour les zonages hors sages-femmes, elles sont issues de l'API geo.api.gouv.fr, qui fournit la population communale la plus récente diffusée par l'Insee. Pour les arrondissements de Paris, Lyon et Marseille, les données ont été manuellement téléchargées sur le site de l'Insee (recensement de la population 2016).
  • Pour le zonage sages-femmes, seule la population féminine est prise en compte. Les données ont été téléchargées manuellement depuis le site de l'Insee (population 2016 par sexe et âge).
  • Les données de population des quartiers prioritaires de la politique de la ville (QPV) ont été téléchargées manuellement sur le site de l'Insee (estimations démographiques 2015).
  • La population des grands quartiers (maille déorgatoire du zonage médecin pour l'île de La Réunion) a été calculée à partir des données de population à l'IRIS, téléchargées depuis le site de l'Insee (bases infracommunales du recensement de la population 2015).

Les données de l'APL sont les dernières données publiées par la DREES sur le site de dataviz dédié, soit le millésime 2018.

NB : Les données de l'APL ne sont pas mises à jour aussi fréquemment que les données géographiques et démographiques des communes.

Légendes et codes couleurs

  • Le tableau de zonage interactif ainsi que la carte interactive et le diagramme "camembert" sont coloriés avec le même code couleur :
    • Pour les sages-femmes & infirmiers : bleu = très sous-doté, vert pâle = sous-doté, vert = intermédiaire, rose = très doté, rouge = sur-doté ;
    • Pour les médecins généralistes : bleu = hors-vivier, vert = zone de vigilance, rose = zone d'accompagnement, rouge = zone d'intervention prioritaire.
  • La couleur des lignes du tableau rappelle le cadre national :
    • Pour les sages-femmes & infirmiers : bleu = très sous-doté, vert pâle = sous-doté, vert = intermédiaire, rose = très doté, rouge = sur-doté ;
    • Pour les médecins généralistes : bleu = hors vivier, vert = vivier, rouge = cadre national.
  • Des cercles rouges autour des boutons radio (cases d'option) rappellent le zonage pris dans le dernier arrêté (téléchargeable dans Documents > Fichier réf. zonage médecin/SF/IDE) ou, à défaut, le cadre national.
  • Des cercles bleus autour des boutons radio (cases d'option) indiquent le zonage en vigueur pris dans une région voisine.

Un tableau situé sous le tableau interactif récapitule les zonages qui diffèrent du cadre national.
